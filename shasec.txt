# SHASEC - Architecture Technique v1

## Vision

Shasec est une plateforme d'audit de sécurité qui orchestre plusieurs scanners, normalise leurs résultats, les analyse avec l'IA et génère des rapports professionnels.

Principe :

Scanner → Corrélation → IA → Rapport

---

# Architecture Générale

Qt Desktop
│
REST / WebSocket
│
Shasec Backend (ShaAPI)
│
Shasec Core
│
Plugin Manager
│
Plugins
│
Nmap / Nuclei / ZAP / Discovery / Custom

---

# Frontend

Module :
shasec-desktop

Technologie :
Qt 6 (C++)

Responsabilités :

* Tableau de bord
* Création de scans
* Suivi des scans
* Consultation des findings
* Consultation des rapports
* Export PDF
* Paramètres

Communication :

REST API
WebSocket

Le frontend ne contient aucune logique métier.

---

# Backend

Module :
shasec-backend

Technologie :
ShaAPI (Python)

Responsabilités :

* API REST
* Authentification
* Gestion utilisateurs
* Gestion projets
* Gestion scans
* Gestion rapports
* Communication WebSocket

Le backend agit comme une couche d'exposition.

Il ne réalise aucun scan directement.

# Contrat Plugin

Entrée :

{
"scan_id": "...",
"target": "https://example.com"
}

Sortie :

{
"plugin": "nuclei",
"findings": []
}

Tous les plugins doivent respecter ce contrat.
